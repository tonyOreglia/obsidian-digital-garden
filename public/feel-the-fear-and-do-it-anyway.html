WORK IN PROGRESS

"How to Read a Book" is an expository practical work written by Susan Jeffers. It was first published in 1987.

It is a book about dealing with fear. Specifically the book describes both useful and harmful beliefs about fear; arguing that we can approach fear in a healthy and empowering way with the right set of beliefs.
